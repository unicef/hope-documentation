# Country Report

Reporting component of the HOPE ecosystem. 
This components allows user to produce reports and keep them updated customizing filters and refreshing schedule. 

## Features

- Custom Queries
- Custom Templates
- Multiple formatting outputs:
  - pdf
  - doc
  - xls
  - txt
  - csv
