# Installation

For the sake of simplicity and ubiquity, we have chosen to use Docker and Docker Compose.


## Required Minimum System Resources

We require at least Docker 19.03.6 and Compose 2.19.0.

- 4 CPU Cores
- 32 GB RAM
- 40 GB Free Disk Space
